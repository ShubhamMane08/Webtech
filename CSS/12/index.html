Different Attributes of Display Properties
  • inline used to change a blocklevel element into inline Ex.: display:inline
  • block used to change a inlinelevel element into block Ex.: display:block
  • inline-block used to give both block and inline elemnts property to any elemnt Ex.: dipslay:inline
  • flex used to make all the first generation child elements into inline_block elements, Creating a flex box which is parent element. Ex.: dipslay:flex
