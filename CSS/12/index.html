Display Properties

Different Attributes of Display Properties
  • inline used to change a blocklevel element into inline Ex.: display:inline
  • block used to change a inlinelevel element into block Ex.: display:block
  • inline-block used to change a blocklevel element into inline Ex.: dipslay:inline
  • flex used to change a blocklevel element into inline Ex.: dipslay:flex
